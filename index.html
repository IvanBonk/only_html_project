Робота зі списками

Впорядковані списки

Впорядковані списки будуються за допомогою тегів ol - тип списку та li - елемент списку
В html розрізняють 5 типів впорядкоіваниї списків:

  • впорядковані цифрами (атрибут type="1")
    1. перший елемент
    2. другий елемент
    3. третій елемент
  • впорядковані латинськими малими літерами (атрибут type="a")
    1. перший елемент
    2. другий елемент
    3. третій елемент
  • впорядковані латинськими великими літерами (атрибут type="A")
    1. перший елемент
    2. другий елемент
    3. третій елемент
  • впорядковані римськими малими цифрами (атрибут type="i")
    1. перший елемент
    2. другий елемент
    3. третій елемент
  • впорядковані римськими великими цифрами (атрибут type="I")
    1. перший елемент
    2. другий елемент
    3. третій елемент

Невпорядковані списки

Непорядковані списки будуються за допомогою тегів ul - тип списку та li - елемент списку
В html розрізняють 4 стандартних типи невпорядкованих списків:

  • без жодних позначок (атрибут type="none")
    • перший елемент
    • другий елемент
    • третій елемент
  • позначені диском (атрибут type="disk")
    • перший елемент
    • другий елемент
    • третій елемент
  • позначені колом (атрибут type="circle")
    • перший елемент
    • другий елемент
    • третій елемент
  • позначені квадратом (атрибут type="square")
    • перший елемент
    • другий елемент
    • третій елемент

Списки понять

Списки понять будуются за допомогою тегів dl - стврення списку, dt - поняття та dd - визначення поняття

Робота зі зображеннями

На сторінку зображення добавляють за допомогою тегу img та атрибуту src - який задає адресу
Також використовуються атрибути height для задання висоти,
wight - для задання ширини, align - для вирівнювання по лівому або правому краю

Назву зображення можна вивести разом зі зображенням.
Для цього використовується тег figure разом з включеним тегом figcaption


Робота з таблицями

Таблицю створюємо за допомогою тегу table.
Після цього пишемо тег tr (table row) - початок нового рядка, та включаємо в нього тег td (table data) - комірка даних.
